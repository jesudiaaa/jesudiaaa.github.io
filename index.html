Aquí te explico los puntos clave: OTG: Host vs. Periférico: Cuando un teléfono actúa como host OTG (para conectar un pendrive, teclado, etc.), es el que suministra energía al dispositivo conectado. Esto es lo contrario a cuando el teléfono se carga, donde recibe energía. La mayoría de los teléfonos están diseñados para una u otra función en un momento dado. Es decir, o están recibiendo energía para cargarse o están suministrando energía como host OTG. USB Power Delivery (USB PD): La clave para la simultaneidad: El estándar USB Power Delivery (USB PD) es lo que permite una gestión de energía mucho más flexible a través de los puertos USB-C. Con USB PD, la dirección de la energía ya no es fija, lo que significa que el dispositivo con el puerto (host o periférico) puede negociar para suministrar o recibir energía. Para que un dispositivo móvil pueda cargar y usar OTG al mismo tiempo, debe tener la especificación USB PD implementada de una manera que le permita negociar un "contrato de energía" con un adaptador o hub compatible. Esto permite que el adaptador proporcione energía tanto al teléfono (para cargarlo) como al dispositivo OTG conectado. Implementación del Fabricante (Hardware y Software/Kernel): Xiaomi Redmi 10C: Es muy probable que el Redmi 10C, siendo un teléfono de gama más económica, no tenga una implementación completa del estándar USB PD que permita la carga simultánea y el uso de OTG. Muchos teléfonos de gama media y baja priorizan la funcionalidad básica y pueden carecer de chips controladores USB más avanzados o de la configuración de firmware/kernel necesaria para manejar este escenario dual. Cuando conectas un dispositivo OTG, el Redmi 10C asume el rol de host y suministra energía, lo que impide que reciba carga al mismo tiempo. Honor 10 (COL-L29 con Kirin 970): El Honor 10, con su procesador Kirin 970 (un chip de gama alta en su momento) y su posicionamiento en el mercado, es mucho más probable que sí tenga una implementación más robusta de USB PD y controladores USB que le permitan negociar la entrega de energía de forma más inteligente. Esto significa que puede ser capaz de recibir carga de un adaptador externo mientras actúa como host OTG para otro dispositivo. Honor/Huawei (Honor era una submarca de Huawei en ese entonces) tiende a incluir estas funcionalidades más avanzadas en sus dispositivos de gama alta. Tipos de adaptadores/cables OTG: Existen cables y hubs OTG especiales que tienen un puerto adicional para conectar un cargador externo. Estos están diseñados para intentar "inyectar" energía al teléfono mientras está en modo OTG. Sin embargo, para que funcionen correctamente, el teléfono debe tener el hardware y el software (kernel) adecuados para reconocer y gestionar esta entrada de energía mientras actúa como host. Si el Redmi 10C no tiene el soporte a nivel de hardware/kernel para el "modo ACA" (Adaptador de Carga de Accesorios) o USB PD de forma completa, un cable OTG con carga podría no funcionar como esperas, o solo cargar el dispositivo sin permitir el OTG, o viceversa. En resumen: La diferencia radica en las capacidades inherentes de hardware y la implementación de software (kernel de Android y controladores USB) que cada fabricante decide incluir en sus dispositivos. Los modelos más económicos, como el Redmi 10C, a menudo omiten estas características para reducir costes.
